# Arkanoid Mobile

A mobile-friendly version of the classic Arkanoid game, built with HTML5 Canvas, CSS, and JavaScript.

## Features
- Touch controls
- Brick-breaking gameplay
- Win and Lose screens
- Sound effects
- Responsive design

## How to Play
- Tap "Start" to begin
- Move the paddle by dragging left/right
- Break all the bricks to win
- Miss the ball and you lose!
